Aprendizaje de Etiquetas Básicas de HTML, Atributos y Valores

Dado al acontecimiento que ha vivido el mundo por la pandemia global Covid19, el ser humano busca métodos de superación, entre ellos el diseño de sitios web que puedan facilitar las actividades cotidianas; ejemplos: comunicaciones, compras, información, entre otros. Por ello la empresa de desarrollo web HtmlDesing presento el siguiente proyecto; que dispone información del conocimiento básico de Html5.

Lenguaje HTML

HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML.
